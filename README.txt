Black Wolf is a modern cross-platform chess gui, able to load engines using the UCI protocol.
Currently hardcoded to use Stockfish - http://stockfishchess.org/

Source code may be found at https://github.com/Kojirion/blackWolf

Libraries used:
SFGUI - http://sfgui.sfml-dev.de/
SFML - http://www.sfml-dev.org/
Thor - http://www.bromeon.ch/libraries/thor/index.html
PStreams - http://pstreams.sourceforge.net/
Boost - http://www.boost.org
